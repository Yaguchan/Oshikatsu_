# Oshikatsu_
・顔の切り出しは[YOLOv8-face](https://github.com/akanametov/yolo-face)を使用  
・顔識別は[facenet-pytorch](https://github.com/timesler/facenet-pytorch)を使用


## 実行
設定の項目を埋めて実行
```
bash run.sh
```
